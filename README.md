# Smart Citizen Service Request System

A web application for citizens to report community issues.

## Project Structure
- `frontend/` - User interface (HTML, CSS, JavaScript)
- `backend/` - Node.js/Express REST API
- `database/` - MySQL database schema

## Quick Start
1. Frontend: Open `frontend/index.html` in browser
2. Backend: `cd backend && npm install && npm start`
3. Database: Import `database/schema.sql`
